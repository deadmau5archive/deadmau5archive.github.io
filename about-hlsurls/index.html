HLS URLs


These are links to HLS playlist files (HLS = HTTP Live Streaming) that Twitch uses to host VODs.
Basically, Twitch cuts up videos in small chunks and these links point to text files that contain information where to find all these chunks.
This is the resource the Twitch player uses to play your VOD. But you can use any other player that supports HLS.
For example: In VLC media player, go to Media ▶ Open Network Stream (Ctrl + N) and put one of the HLS URLs in there. It will play your video, directly from Twitch's VOD servers.
